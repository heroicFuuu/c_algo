# C Algorithm Templates

This repository is a personal collection of commonly used C/C++ algorithm and data structure templates.  
It serves as a quick reference and starting point when practicing algorithms, solving problems, or preparing for interviews/contests.

## Directory Structure
- `binary_search/` : Binary search implementations
- `bitwise/` : Bitwise tricks and utilities
- `fib_heap/` : Fibonacci Heap
- `headers/` : Common header examples
- `interview/` : Interview coding question
- `leetcode_design/` : Design-related solutions from LeetCode
- `os/` : OS-related programs
- `priority_queue/` : Priority queue implementations
- `rb_tree/` : Red-Black Tree
- `sort/` : Sorting algorithms
- `struct/` : Structure-related examples
- `utilis.cpp` / `utilis.h` : Utility functions

## Usage

Each module is self-contained and can be compiled individually using `g++`.  
Follow these steps:

### Example: Compile and run **Binary Search**
#### Step 1: move into the module directory
```bash
cd binary_search
```
#### Step 2: compile the source file
```bash
g++ -std=c++17 -O2 -Wall binary_search.cpp -o a.out
```
#### Step 3: run the program
```bash
./a.out
```
